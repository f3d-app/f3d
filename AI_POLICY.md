# AI Policy

F3D maintainers believe that generative AI is a special tool that require special care, in that regards, if you use Generative AI in the context of F3D, you need to:

- **Disclose that you are using AI**
- **Use it in a precise way, to help with precise tasks**
- **Never let it think or talk for you**

Here are different example usages and how acceptable it is:

1. _I use AI to do research._

This is ok, this is your own internal process.

2. _I use AI to do research and share the results with others._

You MUST disclose the results came from an AI especially if
no other sources corroborate the results of that research.
The model used should be disclosed as well.
If you do not disclose, you will be warned and then BANNED on repeated offenses.

3. _I use AI to do write in github issues and pull request, chat, and discuss with others._

This is unacceptable and you will be warned and then BANNED on repeated offenses.
Use your own voice.

4. _I use AI to improve written text in communication channels because english is not my first language._

This is acceptable as long as it is used just as a correction tool, not a writing tool,
but this is really not needed, you will not be judged on the quality of the text.

5. _I use AI to generate example code then rewrite it from scratch for my own needs when I contribute code._

This is acceptable and doesn't need to be disclosed as long as no AI code is copy/pasted.

6. _I use AI to generate code, test, code comments, documentations and anything that is commited into a git repository._

This must be disclosed in the pull request description, with details on which part of the code was generated and
which model were used. If you do not, you will be warned and then BANNED on repeated offenses.

Generated code will be considered to be written by you and under your own copyright, as any code you write, the commit author should reflect that.

You are expected to have reviewed and corrected the code thoroughly and should be able to answer any questions about
what the code does and why it was written this way. If not, you will be warned and then BANNED on repeated offenses.

If you are a mentee in the [F3D mentoring program](doc/dev/13-MENTORING.md), you should not use generative AI.
If you do, you will be warned and then kicked from the mentoring program on repeated offenses.

7. _I use AI to vibecode my contributions, with no reviews or modifications, or I use agentic AI to generate pull requests and issues._

This is NOT acceptable, if you do so, you will be warned and then BANNED on repeated offenses.
