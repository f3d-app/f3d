# Roadmaps and Release Cycles

F3D is a community driven project and we try to provide as much information as we can about the upcoming features
and release expectations. Please keep in mind while we may try to fulfill what was planned, some features may not be available
as planned and releases can be delayed.

Roadmap for the upcoming work is available in a [dedicated issue](https://github.com/f3d-app/f3d/issues/3108) and loosely mirrored by the [github project](https://github.com/orgs/f3d-app/projects/2).
This roadmap is discussed publicly on [discord](https://discord.f3d.app) and is bound to evolve over time.

If you want to get involved, please feel free to join the [discord](https://discord.f3d.app), comment on the issues to get assigned to them and contribute to the issues that matters to you!

## 4.0.0

Our next planned release is a major release, 4.0.0 planned for October 2026.
This [milestone](https://github.com/f3d-app/f3d/milestone/16) contains the work we would like to put into, but things may shift a bit.

## Release cycle

F3D release cycle is one minor release every three months, in January, April, July and October.
A minor release may be skipped before a major release.
